Commands

/save [notes]
Classify, embed, and store one or more items. Claude extracts entries, assigns type/status/tags/doer, generates 512-dim embedding, inserts to Supabase. If doer=ai, the executor picks it up within 60 seconds.
/save Meeting with Dave re: fire doors at 42 Friarn. Quote due Friday. John to arrange access.
/brief [topic]
Semantic search the brain for a topic. Embeds your query, searches items via pgvector, and loads matching core_memory entries (identity, contacts, properties). Returns the top matches as a context block you can paste into a Claude session.
/brief fire doors
/zoom [topic]
Cosmic Zoom analysis. Runs a 4-quadrant framework (micro-concrete, micro-speculative, macro-concrete, macro-speculative) against the brain, then surfaces the tension where micro and macro frames contradict.
/zoom Should I sell 42 Friarn?
/audit
Run the self-improving brain audit. Gathers 12 corpus diagnostics plus security posture checks, sends to Claude for analysis, returns a health score (0-100) and ranked findings. Results stored back in the brain.
/audit
/review
Full weekly review. Generates a summary of brain activity, then runs Cosmic Zoom on the top macro items from the past 7 days. Also fires automatically Sunday 9am UTC.
/review
/remind [what] [when]
Set a reminder with natural language. Parses the date via Claude, stores a scheduled item, and fires it as a WhatsApp message when due (checked every 15 minutes).
/remind call Kieran about CGT tomorrow 2pm
/status
Instant brain health snapshot. Shows active item count, inbox, focus, missing embeddings, AI queue, reminders, and last WhatsApp capture age. No Claude call — pure data, fast.
/status
/coho
Instant COHO portfolio snapshot via API. Shows occupancy percentage, void rooms by property, and open maintenance count. No Claude call — direct API query.
/coho
/burkeman
Sunday philosophical reflection. Queries the week's brain activity, focus items, and core memory, then generates a short, honest reflection on what truly matters. Also fires automatically Sunday 6pm UTC.
/burkeman
/mission
Three-agent relay pipeline. List pending missions, start one, or check active mission progress. Missions run across Eyes (Gemini), Brain (Claude), and Hands (Cowork) via Supabase relay.
/mission start 1
/help
Lists all available commands with one-line descriptions.
/help
(any text)
Chat mode. Conversational RAG against the brain. Embeds your message, retrieves matching items, answers based only on what's in the brain. Suggests /save if context is thin.
What do I know about HMO Article 4?
(voice note)
Transcribes via OpenAI Whisper, then routes the transcript to chat mode.

How /save works

1

WhatsApp message hits Twilio webhook → Railway agent receives it

2

Claude Sonnet classifies: type, title, content, tags (incl. property slugs), doer (human/ai), 3 next_actions

3

OpenAI generates 512-dim embedding from the content

4

Item inserted to Supabase items table with status=inbox

5

If doer=ai → trigger sets execution_status=pending → executor claims within 60s → handler runs → result via WhatsApp

Doer classification

doer: human

Tasks requiring Alan's physical presence, a phone call, a financial decision, a signature, a viewing, or access to something only he has.

doer: ai

Tasks an AI agent can execute autonomously — web searches, property portal lookups, price comparisons, data queries, drafting emails/letters, calculations, research, summarising.

AI executor handlers

TagHandlerWhat it does
researchresearchHandlerClaude Sonnet research, writes result to item
draftdraftHandlerClaude Sonnet drafting (emails, letters, notices)
embedembedHandlerGenerates 512-dim embedding (silent)
notifynotifyHandlerSends WhatsApp alert with task content
cleanupcleanupHandlerDedup + archive old done items
(none)genericHandlerClaude catch-all for untagged tasks
Executor retries up to 3 times with exponential backoff (60s, 300s, 900s). After 3 failures: status=failed, WhatsApp alert sent. Safety rails: no financial transactions, no destructive DB ops, only Claude/OpenAI/Twilio APIs.
